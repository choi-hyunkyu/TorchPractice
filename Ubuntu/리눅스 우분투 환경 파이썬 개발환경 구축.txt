# 리눅스 우분투 환경 파이썬 개발환경 구축

'''
개발자 모드 설정
'''
1. 윈도우 검색창에서 '개발자 설정'검색 후 개발자 모드 활성화
2. 제어판/프로그램/프로그램 및 기능/Windows 기능 켜기끄기/Linux용 Windows 하위 시스템 선택
3. 컴퓨터 다시 시작

'''
우분투 기본 세팅
'''
1. sudo apt-get update
2. sudo apt-get upgrade
3. sudo apt-get dist-upgrade
4. sudo apt-get install build-essential python-dev python-setuptools python-pip python-smbus
5. sudo apt-get install libncursesw5-dev libgdbm-dev libc6-dev
6. sudo apt-get install zlib1g-dev libsqlite3-dev tk-dev
7. sudo apt-get install libssl-dev openssl
8. sudo apt-get install libffi-dev

'''
파이썬 설치
'''
1. wget https://www.python.org/ftp/python/3.8.6/Python-3.8.6.tgz
2. ls 명령어 클릭 후 Python-3.8.6.tgz 파일 확인
3. tar xvfz Python-3.8.6.tgz
4. cd 명령어를 통해 Python-3.8.6 폴더 내부로 진입
5. ls 명령어 클릭 후 configure 파일 확인
6. ./configure -> make -> sudo make install
7. Python3 -V

'''
pip3 설치
'''
sudo apt-get install python3-pip
sudo apt-get install python3-setuptools

'''
라이브러리 설치
'''
sudo pip3 install numpy
sudo pip3 install pandas
sudo pip3 install matplotlib

'''
설치 확인
'''
sudo pip3 list

'''
파이토치 설치
'''
1. CPU: pip3 install torch==1.7.0+cpu torchvision==0.8.1+cpu torchaudio==0.7.0 -f https://download.pytorch.org/whl/torch_stable.html

2. GPU: pip3 install torch torchvision
